402ci Chevy LS2 Engine Build Upgrades

Our 402ci Chevy LS2 engine comes dyno tested and will make 550hp using the stock LS3 intake manifold and 90mm throttle body but there are some power gains as well as different looks to be had with the intakes listed below.

Intake Upgrades:

  • Fast LSxR Extra 10-15hp
  • MSD Air Force Extra 15-20hp
  • Holley Sniper Sheet Metal Extra 10-15hp
  • Holley Hi Ram Extra 20hp


Throttle Body Upgrades:

  • Nick Williams 102mm DBW
  • Holley 92mm Cable
  • Holey 102mm Cable  


This engine comes standard with factory GM valve covers and below is a list of the most popular valve covers that we offer as upgrades, if you do not see what you want just ask and we can get it for you.


Valve Cover upgrades:

  • Holley Valve covers
  • Callies Billet Valve Covers


Front Runner Kits: (If you are Retro Fitting an LS Engine into a Hot Rod we offer the following options)

  • Holley Front Drive 
  • Eddie Motorsports


EFI System options:  

  • Holley HP EFI System (this is for cable driven throttle bodies only)
  • Holley Dominator EFI Systems (can run electronic DBW throttle bodies and electronic auto transmissions) 

 

Ignition System options:

  • GM LS7 D514A coil kits


Header Options: 

  • American Racing Headers
  • Kooks Racing Headers


Motor Mount Options:

  • Holley Swap motor mounts


Transmission Options:

  • RPM Transmission 4L60E
  • RPM Transmission 4L80E
